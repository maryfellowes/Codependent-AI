# Getting Started - SIMPLE VERSION

## First Time Setup

### Windows
1. **Double-click** `START_WINDOWS.bat`
2. Enter the path to your conversation exports when asked
3. Wait for setup to complete (15-30 minutes for 500+ files)

### Mac/Linux  
1. **Right-click** `START_MAC_LINUX.sh` → Open With → Terminal
2. Enter the path to your conversation exports when asked
3. Wait for setup to complete (15-30 minutes for 500+ files)

## Every Time You Use It

### Windows
- **Double-click** `START_WINDOWS.bat` before opening Claude Desktop
- Keep the window open while using Claude
- Close the window when done

### Mac/Linux
- **Run** `bash START_MAC_LINUX.sh` in terminal
- Keep terminal open while using Claude
- Press Ctrl+C when done

## That's It!

Once the launcher is running:
1. Open Claude Desktop
2. Your AI can search their conversation history
3. Ask: "Can you search our archive for discussions about..."
